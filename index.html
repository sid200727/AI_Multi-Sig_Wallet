HOW IT WORKS
1
Boolean Policy Engine
Built on the boolpolicy package from Hyperledger fabric-token-sdk PR #1597. Expressions like ($0 AND $1) OR $2 are evaluated against agent signatures.
2
AI Agent Personalities
Each agent is a Claude AI instance with a distinct personality. Alice is cautious, Bob is pragmatic, Charlie is aggressive. They read each other's responses and react.
3
Signature Collection
After all required agents respond with SIGN or REFUSE, the policy expression is evaluated. If satisfied, the token transfer executes. If not, it's rejected.
